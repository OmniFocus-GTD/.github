# OmniFocus — Portable Professional GTD Task Management System

## About OmniFocus

OmniFocus provides a complete GTD (Getting Things Done) task management environment for users managing complex projects. OmniFocus combines hierarchical projects, contexts, perspectives, and a built‑in review system. OmniFocus supports defer dates, due dates, repeat rules, and estimated durations. OmniFocus enables users to capture tasks quickly, organize by project and context, and focus on what matters now. OmniFocus runs on desktop platforms. OmniFocus is trusted by project managers, executives, entrepreneurs, developers, and productivity professionals worldwide.
